# Myst
Aplicación móvil integral que permita registrar, analizar y comprender el ciclo menstrual y sus síntomas, incorporando inteligencia artificial para acompañar y facilitar la comunicación entre médico-paciente en una consulta, promoviendo el bienestar físico y emocional de las usuarias.

## Objetivos específicos:
Implementar un sistema de registro de ciclo menstrual con seguimiento de fases y síntomas físicos, emocionales y hormonales.

Crear un panel de tendencias con visualización de correlaciones entre hábitos, emociones y síntomas.

Desarrollar un chat asistido con IA basado en NLP (GPT) que ayude a registrar síntomas, detectar señales de alerta y fomentar hábitos saludables.

Implementar modelos de machine learning (Random Forest, Hierarchical Clustering, CatBoost o alternativo) para predicción de síntomas y análisis de patrones clínicamente útiles.
*    Incorporar módulos complementarios:
*    Calendario integral de salud.
*    Diario digital.
*    Correlación con resultados de laboratorio.
*    Modo acompañante (compartir reportes con médicos o familiares).

Garantizar privacidad y cumplimiento normativo bajo ISO/IEC 27001 y la LFPDPPP.
